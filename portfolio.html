Day Scheduler

For this solo project I created a simple scheduler from 9-5. The colors of the input areas change with the current time. You can save the info for later or you can clear all inputs. GitHub Repo

Weather Dashboard

For this solo project I created a simiple weather search. Search a city to get the curent weather as well as a 5-day forecast. The cities you have searched will be saved below the search bar if you wish to see them later. You can also clear the current and all past searches. GitHub Repo

Diet Watchers

For this team project I worked with 3 other individuals to create a nutrition and recipe search. You can search an ingredient or common food to get the nutrition details of that food. Or you can search for recipes of the same item. Your most recent recipe search will be saved. You can also clear the nutrition details and your past recipe search. GitHub Repo
